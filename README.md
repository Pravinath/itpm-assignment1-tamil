## Project Description
This project contains Playwright-based automated test cases for the Tamil transliteration website:

https://tamil.changathi.com/

The test suite validates:
- Positive Thanglish to Tamil conversions
- Negative input scenarios
- UI behavior (clear/reset)
- Edge cases such as symbols, slang, spacing, and abbreviations


## Prerequisites
- Node.js (v18 or above recommended)
- npm (comes with Node.js)
- Git

## Installation Steps

1. Clone the repository
   ```git clone https://github.com/Pravinath/itpm-assignment1-tamil.git```


2. Navigate to the project folder
   ```cd itpm-assignment1-tamil```


3. Install dependencies
   ```npm install```


4. Install Playwright browsers
   ```npx playwright install```


## Running the Automated Tests
   ```npx playwright test```

## Viewing Test Results
   ```npx playwright show-report```
